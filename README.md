# Movie App i React

### Ett grupprojekt som jag kopierat till eget repo

![screenshot av appen](./images/Skärmbild%20NeonCinema.pngskärmbild NeonCinema.png)

Målet med uppgiften var att bygga en webbapp som liknar IMBD, där användaren ska kunna söka efter filmer, se trailers och skapa egen watchlist.

Appen hämtar data från externa API och fokuserar på komponentbaserad struktur i React. Vi arbetade agilt med Git Project

---

### Funktioner

- Söka filmer via TMDB API
- Visa topplista och kategori
- En karusell med trailers (trailers funkar ej)
- Klicka på en film för mer detaljerad info.
- Lägg till och ta bort filmer i en watchlist.

### Design

Ett krav i uppgiften var att ta fram design med valfri AI och Figma Make. Vi valde en design med ett retro-inspirerat tema. Våra tankar:

- Mörk bakgrund
- Neon skyltar
- Bio-känsla
- tydlig färgklick

### Tekniker

- React (komponenter, props, useState, useEffect)
- API-anrop
- Routing
- CSS

### Min roll i projektet

I detta projekt har jag främst arbetat med:

- Skapat layout för filmkorten med CSS Grid i REact.
- Stylat flera delar av applickationen som karusell, watchlist och footer.
- Säkerställt grundläggande tillgänglighet med hjälp av WAVE
